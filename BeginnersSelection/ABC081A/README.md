# ABC081A - Placing Marbles
実行時間制限: 2 sec / メモリ制限: 256 MB

## 問題文
すぬけ君は 1,2,3 の番号がついた 3 つのマスからなるマス目を持っています。 各マスには 0 か 1 が書かれており、マス i には 
si が書かれています。

すぬけ君は 1 が書かれたマスにビー玉を置きます。 ビー玉が置かれるマスがいくつあるか求めてください。

### 制約
s1,s2,s3 は 1 あるいは 0

### 入力
入力は以下の形式で標準入力から与えられる。

s1s2s3

### 出力
答えを出力せよ。

### 入力例 1 
101

### 出力例 1 
2
マス 1,3 にビー玉が置かれます

### 入力例 2 
000

### 出力例 2 
0ビー玉はどのマスにも置かれません
